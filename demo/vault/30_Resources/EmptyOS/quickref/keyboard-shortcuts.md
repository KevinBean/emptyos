## Demo navigation

- `/` — focus the search bar (anywhere)
- `Ctrl+K` / `Cmd+K` — command palette
- `g` then `h` — go to Hub (home)
- `g` then `t` — go to Tasks
- `g` then `j` — go to Journal
- `?` — show shortcut help

## Try it

Press `/` now and type `arvo` — you'll see the Music Library has the Arvo Pärt entry.
