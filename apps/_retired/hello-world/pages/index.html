Hello, world
Minimal reference app. Submits your name to POST /hello-world/api/greet, which routes through the think capability and emits hello-world:greeted.
What this app demonstrates
manifest.toml → id, capability requirement, web prefix, emitted events
BaseApp → @cli_command, @web_route, self.think, self.emit
pages/index.html → theme.css + eos-components, EOS.post, EOS_UI.toast
