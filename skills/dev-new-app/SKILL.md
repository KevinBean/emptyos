# New Mini-App Scaffold

Create a new mini-app in the Home Portal ecosystem with all required components.

## Steps

1. **Create HTML page** in `pages/` with standard template (shared nav, toast, modal imports)
2. **Add backend route** in `serve.py`
3. **Register in app config** with icon and group in `apps.json`
4. **Kill any existing server process** on the port
5. **Restart server** and verify the new page loads
